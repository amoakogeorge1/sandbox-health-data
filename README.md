## Description
This is a **sandbox project** for learning health data science workflows.  
It allows beginners to practice **Python, Jupyter notebooks, and data analysis** without needing real datasets.  

---

## Folder Structure
- `data/` – store datasets (fake or real)  
- `notebooks/` – Jupyter notebooks for coding practice  
- `environment.yml` – defines Python environment and libraries  
- `README.md` – project description
